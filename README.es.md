## Preparando

```sh
git clone https://github.com/martinval11/dolar-argentina
cd dolar-argentina
pnpm install
```

## Scripts

- `pnpm dev` - Despliega un entorno de desarrollo
- `pnpm build` - Construye un entorno de producción
- `pnpm test` - Ejecuta los tests con Cypress
